You get help in every stage of your product

Developing app with a new backend

You want to start working on your app, but the backend is still behind in functionality. You either wait, or mock whole responses. With Proxytool, you can add any data to the response, to make it whole.

Read more

Zero-downtime when replacing the backend

Your obsolete backend is replaced with a fancy new one. Changing the API URL, your app just crashes until everything is updated. With Proxytool, you can map the new API to the old format, enabling you to fix your app call-by-call.

Read more

Testing hard-to-reproduce scenarios

How would your app behave if the user does something weird? Would it break if it gets an error? With Proxytool you can set up very specific datasets, so you can check how your app behaves in different scenarios.

Read more

Introducing Proxytool, your new best friend for mocking

Mock/change only a part of the request/response. For example, you can add an extra header to the request, like an auth token. Or you can add an icon to every element in the response list.

Learn more

Mock an entire response. When your backend is not ready, just mock like it's working

You want to achieve something custom tailored,based on the parameters? Don't worry, we got you covered. You can also make requests from the custom code, so you can get data from other APIs.

Learn more
